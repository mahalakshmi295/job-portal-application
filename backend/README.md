# Professional Job Portal Backend

This backend is built with Spring Boot and provides REST APIs for:

- User management (with role-based access)
- Job management
- AI-based skill gap analysis
- Resume quality scoring
- Job match percentage
- Real-time notifications
- Bias-free candidate ranking
- Analytics dashboard
- OTP-based login
- Chatbot for job assistance

## How to run

1. Make sure you have Java 17+ and Maven installed.
2. In the backend directory, run:
   ```
   mvn spring-boot:run
   ```
3. The backend will be available at http://localhost:8080

## API Endpoints

- `/api/health` - Health check
- `/api/users` - User management
- `/api/jobs` - Job management
- `/api/skill-gap` - Skill gap analysis
- `/api/resume-score` - Resume scoring
- `/api/job-match` - Job match percentage
- `/api/notifications` - Notifications
- `/api/analytics` - Analytics dashboard
- `/api/chatbot` - Job assistance chatbot
- `/api/ranking` - Bias-free candidate ranking
- `/api/auth/otp-login` - OTP login
- `/api/auth/verify-otp` - OTP verification
