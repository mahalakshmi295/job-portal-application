# 💼 Job Portal Application (Full Stack AI Powered)

A full-stack Job Portal application built using **Java Spring Boot (Backend)** and **React + TypeScript (Frontend)**.  
It provides AI-powered job search, resume analysis, and career guidance features.

---

## 📁 Project Structure

- 🎨 Frontend: React + TypeScript
- ⚙️ Backend: Java Spring Boot REST APIs
- 🗄️ Database: MySQL / H2 (for testing)

---

## 🚀 Key Features

### 🤖 AI Features
- AI Skill Gap Analysis using job market data
- AI Chatbot for career guidance (fallback + intelligent responses)
- Resume Quality Scoring system
- Smart Job Matching with percentage score

---

### 🌐 Job Features
- Real-time job search using external APIs
- Job details and salary insights
- Location-based job suggestions
- Trending job analytics

---

### 🔐 Security Features
- OTP-based email login authentication
- JWT-based secure authentication
- Role-based access control (Admin/User)

---

### 📊 Dashboard Features
- Job market analytics
- Application tracking system
- Skill demand insights

---

## 🛠️ Tech Stack

**Frontend:** React, TypeScript, HTML, CSS  
**Backend:** Java, Spring Boot  
**Database:** MySQL  
**APIs:** OpenAI, RapidAPI, Affinda, ipapi  
**Tools:** Git, GitHub, Maven, Postman, VS Code
